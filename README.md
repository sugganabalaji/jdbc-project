# JDBC PostgreSQL Project

## 📌 Overview
This project demonstrates how to connect a Java application to a PostgreSQL database using JDBC. It covers basic CRUD operations and serves as a learning resource for database connectivity.

## ⚙️ Features
- Connect to PostgreSQL using JDBC
- Create, Read, Update, Delete (CRUD) operations
- Student management (SID, SName)
- Prepared statements for secure queries
- Exception handling and modular design

## 🛠️ Requirements
- JDK 17+ (or your preferred version)
- PostgreSQL installed locally
- PostgreSQL JDBC driver (e.g., `postgresql-42.x.x.jar`)
